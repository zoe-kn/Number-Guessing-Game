# Number Guessing Game

A simple browser-based number guessing game written in JavaScript.  
The player attempts to guess a randomly generated number, and the interface provides feedback such as “higher” or “lower”.  
A confetti animation appears when the correct number is guessed.

---

## Features

- Random number generation between 0 and 100  
- Interactive input field for guesses  
- Dynamic feedback: higher / lower / correct  
- Attempt counter  
- Confetti animation using JSConfetti  

---

## How It Works

The game generates a random number when the page loads.
The User enters a number into the input field and the game tells the user if the number is higher or lower.
